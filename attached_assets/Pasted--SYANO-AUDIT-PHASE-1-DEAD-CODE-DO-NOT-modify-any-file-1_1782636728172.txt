# ════════════════════════════════
# TASK 1 — UNUSED IMPORTS
# ════════════════════════════════
Read every .ts and .tsx file in scope.
For each file, find every import that meets ALL of these:
- The imported name never appears in the file body after the import line
- It is not a type-only import used in a JSX prop type
- It is not a side-effect import (import 'some-css' is intentional)

Pay special attention to:
- lucide-react icon imports (commonly 10 imported, 3 used)
- TanStack Query: useQuery, useMutation, useQueryClient, useInfiniteQuery
- Zod: z, ZodError, ZodSchema imported but no schema defined in file
- Drizzle operators: eq, and, or, desc, asc, sql imported but unused
- React: useState, useEffect, useCallback, useMemo, useRef imported but unused

For each finding write EXACTLY this format — nothing else:
---
FILE: artifacts/marketplace/src/components/Example.tsx
LINE: 3
IMPORT: import { useState, useCallback } from 'react'
UNUSED: useCallback
SAFE_TO_REMOVE: YES
---

# ════════════════════════════════
# TASK 2 — DEAD VARIABLES
# ════════════════════════════════
Read every .ts and .tsx file in scope.
Find const/let declarations where the variable is:
- Declared and assigned a value
- Never read anywhere in the same file
- Not exported for use in other files

Also find:
- Destructured values never used: const { a, b, unusedC } = obj
- useState pairs where the state value is set but never read in JSX
- useRef where .current is never accessed after creation

DO NOT flag:
- Variables starting with _ (intentionally unused by convention)
- Variables exported from the file (might be used elsewhere)
- Variables used only in return statements or JSX

For each finding:
---
FILE: artifacts/api-server/src/routes/products.ts
LINE: 45
CODE: const tempData = await db.select()...
REASON: Assigned but never read or returned
SAFE_TO_REMOVE: YES / REVIEW_FIRST
---

# ════════════════════════════════
# TASK 3 — DEAD FUNCTIONS
# ════════════════════════════════
Read every .ts and .tsx file in scope.
Find functions (including arrow functions) that are:
- Defined in the file
- Never called within the same file
- Never exported from the file

Also find custom hooks in any hooks/ folder that:
- Are never imported by any other file in the codebase

For React-specific dead code:
- useCallback wrapping a function never passed as a prop or called
- useMemo computing a value never used in JSX or logic
- useEffect with an empty dependency array that does nothing
  (empty body, or only sets state that is never read)

For each finding:
---
FILE: artifacts/marketplace/src/components/ProductCard.tsx
LINE: 67
CODE: const handleOldClick = useCallback(() => { ... }, [])
REASON: handleOldClick never attached to any element or passed as prop
SAFE_TO_REMOVE: YES / REVIEW_FIRST
---
