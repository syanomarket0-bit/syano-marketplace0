# TASK 3 — VITE PROXY VERIFICATION
# ══════════════════════════════════════
Read: artifacts/marketplace/vite.config.ts

- /api/* is proxied to localhost:8080 (or process.env.API_PORT)
- The proxy config has NOT been removed or modified
- No other proxy rules conflict with the API proxy

Also find:
- Any fetch('/api/...') call in artifacts/marketplace/src/ that
  uses a hardcoded port (e.g. fetch('http://localhost:8080/api/...')
  instead of fetch('/api/...') — these bypass the Vite proxy

For each finding:
