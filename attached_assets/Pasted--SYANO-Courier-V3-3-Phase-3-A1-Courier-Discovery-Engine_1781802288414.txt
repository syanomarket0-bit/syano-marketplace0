```text
Haversine Formula
```

for geographic distance.

Do NOT use fake calculations.

Do NOT use rough estimates.

Use real latitude/longitude distance.

---

```ts
calculateDistanceKm(
  lat1,
  lng1,
  lat2,
  lng2
)
```

Return:

```ts
number
```

# DISCOVERY SERVICE

```ts
findNearestCouriers(

  pickupLng,
  limit = 3
)
```

```text
Load Available Couriers

↓
```
