# A2.3 OFFER LIFECYCLE ENGINE

Core engine:

offerMissionToNearestCouriers()

Input:

missionId

Process:

1.

Load mission

2.

Determine pickup coordinates

3.

Call:

findNearestCouriers()

4.

Take nearest 3 couriers

5.

Create offer for first courier

6.

Start countdown

Default:

5 minutes

Store:

expiresAt

Mission state becomes:

OFFERING

---

# A2.4 ACCEPT FLOW

When courier accepts:

Validate:

* offer exists
* offer still active
* mission not assigned
* courier still available

Then:

Offer:

ACCEPTED

Mission:

ASSIGNED

Courier:

BUSY

mission.courierId = accepted courier

Cancel remaining offers automatically

---

# A2.5 DECLINE FLOW

When courier declines:

Offer:

DECLINED

Store:

respondedAt

Immediately:

Offer next courier

If another courier exists.

---

# A2.6 EXPIRE FLOW

If countdown expires:

Offer:

EXPIRED

Automatically:

Offer next courier

No manual intervention.

---

# A2.7 ALL COURIERS FAILED

If:

all 3 couriers

declined

or expired

Then:

Mission:

UNASSIGNED

Assignment status:

FAILED_AUTO_ASSIGNMENT

Visible in admin dashboard.
