# CRITICAL GOAL

The SYANO mobile application must become a true mobile version of the existing web marketplace.

The objective is:

A user should not be able to say:

"This feature exists on web but not mobile."

or

"This screen looks different from web."

or

"This workflow behaves differently."

Mobile must mirror web behavior, structure, visual hierarchy, business logic, translations, permissions, and data flow.
