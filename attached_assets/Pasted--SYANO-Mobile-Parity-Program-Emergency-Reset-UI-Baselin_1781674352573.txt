The mobile application is not currently a true mobile version of the web platform.

> Mobile App = Native wrapper around the exact SYANO experience.

A user should not feel they are using a different product.

Right now there are clear signs that mobile contains:

* outdated screens
* old designs
* legacy layouts
* components that no longer exist on web
* duplicated UI logic
* custom mobile implementations that diverge from web

Example:

Current login screen shown in mobile preview is NOT the current SYANO login experience.

The web version has evolved significantly.

The mobile app is visually behind.

This is unacceptable.
