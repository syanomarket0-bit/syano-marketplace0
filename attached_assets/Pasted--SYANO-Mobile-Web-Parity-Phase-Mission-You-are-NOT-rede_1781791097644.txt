# SYANO Mobile Web-Parity Phase

## Mission

You are NOT redesigning the mobile app.

You are NOT improving the mobile app.

You are NOT creating a mobile interpretation.

Your ONLY goal is:

**Make the mobile homepage a pixel-faithful React Native implementation of the current web homepage.**

The web homepage is the source of truth.

The mobile homepage must become a direct implementation of the web version.

---

# CRITICAL RULE

Do NOT continue to the next item until the current item is verified as matching the web implementation.

Work sequentially.

If item #1 is not fully matched:

* STOP
* Fix it
* Recheck it
* Then continue

Never skip ahead.

Never partially complete a section.

Never say "close enough".

Never approximate.

---

# SOURCE OF TRUTH

Always use:

artifacts/marketplace

as the reference implementation.

Everything inside:

* Homepage
* Hero
* Navbar

* Typography
