Your job is to detect and surface data quality issues to sellers and admins. Do not delete any product, user, or order. Do not modify any existing data. Only add detection, flagging, and UI indicators.

WHAT TO DETECT
For Products — flag if any of these are true:

images array is empty or null
description is empty, null, or under 20 characters
description_ar is empty, null, or under 20 characters
name_ar is empty or null
price is 0 or null
stock is 0 (out of stock — not hidden, just flagged)
embedding is null (not yet embedded — affects search quality)

For Stores — flag if any of these are true:

logo is null or empty
description is empty or null
description_ar is empty or null


STEP 1 — BACKEND: Add Data Quality Endpoints
Endpoint 1 — Seller: own product quality
File: artifacts/api-server/src/routes/seller.ts
GET /api/seller/products/quality-report

Requires requireAuth + requireRole('seller')
Returns only the seller's own products that have quality issues
Response shape:

typescript{
  total_products: number,
  flagged_count: number,
  products: Array<{
    id: number,
    name: string,
    name_ar: string,
    issues: string[] // e.g. ['missing_images', 'short_description_ar', 'not_embedded']
  }>
}
Endpoint 2 — Admin: platform-wide quality report
File: artifacts/api-server/src/routes/admin.ts
GET /api/admin/products/quality-report

Requires requireAuth + requireRole('admin')
Returns platform-wide counts and flagged products
Response shape:

typescript{
  total_products: number,
  flagged_count: number,
  flagged_percentage: number,
  breakdown: {
    missing_images: number,
    short_description: number,
    short_description_ar: number,
    missing_name_ar: number,
    zero_price: number,
    out_of_stock: number,
    not_embedded: number
  },
  products: Array<{
    id: number,
    name: string,
    name_ar: string,
    seller_id: number,
    store_name: string,
    issues: string[]
  }>
}
Endpoint 3 — Admin: store quality report
File: artifacts/api-server/src/routes/admin.ts
GET /api/admin/stores/quality-report

Requires requireAuth + requireRole('admin')
Response shape:

typescript{
  total_stores: number,
  flagged_count: number,
  stores: Array<{
    id: number,
    name: string,
    seller_id: number,
    issues: string[] // e.g. ['missing_logo', 'missing_description_ar']
  }>
}
