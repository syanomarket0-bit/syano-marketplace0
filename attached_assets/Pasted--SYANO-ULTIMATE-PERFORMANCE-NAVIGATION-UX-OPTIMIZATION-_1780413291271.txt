# SYANO ULTIMATE PERFORMANCE, NAVIGATION & UX OPTIMIZATION MISSION

You are taking over the Syano Marketplace production codebase.

Syano is a large-scale full-stack marketplace built with:

* React 19
* Vite 7
* Wouter
* Tailwind CSS v4
* TanStack Query
* Orval-generated API hooks
* Express 5
* PostgreSQL
* Drizzle ORM
* Expo Mobile

The application is already working.

Your mission is NOT feature development.

Your mission is to make the application feel dramatically faster, smoother, more responsive, and more polished across the entire platform while preserving 100% existing functionality.

---

## CRITICAL RULES

DO NOT:

* Change business logic
* Change database behavior
* Change authentication flow
* Change seller workflow
* Change customer workflow
* Change admin workflow
* Change API contracts
* Change OpenAPI definitions
* Change permissions
* Change route structure
* Change visual design language
* Remove features
* Rewrite large systems unnecessarily

The application must behave exactly the same after your work.

Only improve:

* Speed
* Smoothness
* Responsiveness
* Rendering efficiency
* Navigation performance
* User experience consistency

If an optimization introduces risk:

DO NOT APPLY IT.

Document it instead.

---

## MAIN OBJECTIVE

Target outcome:

The application should feel premium.

Navigation should feel immediate.

Pages should appear instantly whenever possible.

Returning to previously visited pages should feel nearly instant.

Scrolling should remain smooth even on low-end Android devices.

Desktop interactions should feel as responsive as high-end SaaS applications.

No visible lag after clicks.

No sluggish dashboards.

No delayed menus.

No UI stalls.

No unnecessary loading states.
