# Current Problems

## 1. Product cards overlap the footer

The cards are escaping the normal document flow.

Instead of pushing the page downward naturally, they appear on top of the footer.

This usually indicates one of:

* broken CSS Grid
* broken Flex layout
* missing wrapper
* incorrect parent height
* absolute positioning
* overflow:hidden on parent
* fixed height container
* missing flex-wrap
* wrong display property

Inspect the complete hierarchy:

Page
→ Layout
→ Main Content
→ Products Container
→ Products Grid/List
→ Product Card
→ Footer

Find exactly where the flow breaks.

The footer must always appear AFTER the products section.
