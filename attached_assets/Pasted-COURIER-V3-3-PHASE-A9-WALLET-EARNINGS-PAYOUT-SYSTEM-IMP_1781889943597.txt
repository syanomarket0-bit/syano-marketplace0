# PART A9.1

FINANCIAL DATA MODEL

Create required tables.

---

## courier_wallets

One wallet per courier.

Fields:

* id
* courier_id
* available_balance
* pending_balance
* lifetime_earnings
* lifetime_payouts
* created_at
* updated_at

---

## courier_transactions

Financial ledger.

Fields:

* id

* courier_id

* type

ENUM:

EARNING
ADJUSTMENT
PAYOUT_REQUEST
PAYOUT_APPROVED
PAYOUT_REJECTED

* amount
* balance_before
* balance_after
* reference_id
* reference_type
* description
* created_at

Never delete transactions.

Ledger is source of truth.

---

## courier_payout_requests

Fields:

* id
* courier_id
* amount
* status

ENUM:

PENDING
APPROVED
REJECTED
PAID

* rejection_reason
* approved_by
* approved_at
* paid_at
* created_at
