CRITICAL ENTERPRISE-GRADE ADMIN COMMAND CENTER

This is NOT a simple admin dashboard improvement.

The objective is to transform Syano's admin area into a complete marketplace command center similar to what would be used by Amazon Marketplace, Noon, Trendyol, Etsy, or Shopify Marketplace operators.

The admin dashboard must become the central control system for the entire marketplace.

══════════════════════════════════════
PRIMARY OBJECTIVE
══════════════════════════════════════

The administrator should be able to understand the complete health of the marketplace within seconds.

The dashboard must answer:

* What is happening right now?
* What is growing?
* What is failing?
* Which sellers need attention?
* Which products are performing best?
* Which categories are growing?
* Where are users dropping off?
* Which operational issues require intervention?

The admin should never need to manually inspect multiple pages to understand the state of the platform.
