CRITICAL MOBILE UI & UX AUDIT

This task is NOT about adding new features.

The goal is to make the marketplace feel like a polished, production-ready mobile application.

Mobile experience is the highest priority.

Most users will access the marketplace from mobile devices.

Every screen must be audited and optimized.

══════════════════════════════════════
OBJECTIVE
══════════════════════════════════════

Perform a complete mobile-first UI/UX audit across the entire marketplace.

Identify:

* Layout issues
* Overflow issues
* Alignment issues
* RTL issues
* LTR issues
* Spacing issues
* Typography issues
* Touch-target issues
* Navigation issues
* Visual consistency issues

Do not assume screens are correct.

Audit every page.
