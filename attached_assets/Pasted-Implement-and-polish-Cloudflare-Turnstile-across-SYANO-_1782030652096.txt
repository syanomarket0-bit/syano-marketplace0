# TURNSTILE DESIGN REQUIREMENTS

The current Turnstile widget is too large and visually intrusive.

Redesign its integration so it feels native to SYANO.

Requirements:

* Compact appearance
* Small horizontal rectangle
* Never full-width
* Maximum width: 280–320px
* Auto height
* Centered horizontally
* Consistent spacing with surrounding form fields
* Match SYANO border radius, shadows, spacing, and colors
* No visual dominance over the form

The widget should feel like a small verification element, not a giant component.

Target visual quality:

* Shopify
* Stripe
* Notion
* Linear

# AUTO RESET BEHAVIOR

Implement:

onSuccess:

* Store token

onExpire:

* Clear token
* Reset widget

onError:

* Reset widget

If backend returns:

TURNSTILE_INVALID

Then:

* Clear token
* Auto reset widget
* Show localized error message
* Allow retry without page refresh
