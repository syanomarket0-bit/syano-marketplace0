STEP 2 — DOWNLOAD pytorch_model.bin
─────────────────────────────────────────────
Download the model weights directly into the model folder:

  wget -c --retry-connrefused --tries=10 --timeout=60 \
    -O artifacts/embedding-service/model/pytorch_model.bin \
    "https://huggingface.co/sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2/resolve/main/pytorch_model.bin"

Flags explanation:
  -c = resume if interrupted
  --tries=10 = retry up to 10 times on failure
  --timeout=60 = 60s per connection attempt

If wget fails or is not available, try curl:
  curl -L --retry 10 --retry-delay 5 --continue-at - \
    "https://huggingface.co/sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2/resolve/main/pytorch_model.bin" \
    -o artifacts/embedding-service/model/pytorch_model.bin

If the file is smaller than 400MB: it downloaded incompletely. Delete it and retry.
