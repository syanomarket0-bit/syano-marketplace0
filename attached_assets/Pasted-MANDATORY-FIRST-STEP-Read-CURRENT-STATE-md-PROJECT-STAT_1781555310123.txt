STEP 4 — DOWNLOAD THE EMBEDDING MODEL
─────────────────────────────────────────────
Create the model directory:
  mkdir -p artifacts/embedding-service/model/1_Pooling

Download all model files from HuggingFace.
The model is: sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2

Download config files first (small, fast):
  cd artifacts/embedding-service/model

  curl -L -o config.json \
    "https://huggingface.co/sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2/resolve/main/config.json"

  curl -L -o tokenizer_config.json \
    "https://huggingface.co/sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2/resolve/main/tokenizer_config.json"

  curl -L -o tokenizer.json \
    "https://huggingface.co/sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2/resolve/main/tokenizer.json"

  curl -L -o special_tokens_map.json \
    "https://huggingface.co/sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2/resolve/main/special_tokens_map.json"

  curl -L -o modules.json \
    "https://huggingface.co/sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2/resolve/main/modules.json"

  curl -L -o sentence_bert_config.json \
    "https://huggingface.co/sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2/resolve/main/sentence_bert_config.json"

  curl -L -o sentencepiece.bpe.model \
    "https://huggingface.co/sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2/resolve/main/sentencepiece.bpe.model"

  curl -L -o unigram.json \
    "https://huggingface.co/sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2/resolve/main/unigram.json"

  curl -L -o 1_Pooling/config.json \
    "https://huggingface.co/sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2/resolve/main/1_Pooling/config.json"

Now download the model weights (449MB — the most important file):
  curl -L --retry 10 --retry-delay 5 \
    "https://huggingface.co/sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2/resolve/main/model.safetensors" \
    -o model.safetensors

After download, verify:
  ls -lh artifacts/embedding-service/model/
  Expected: model.safetensors ~449MB, all config files present

If model.safetensors download fails after 3 retries:
  Try pytorch_model.bin instead:
  curl -L --retry 10 --retry-delay 5 \
    "https://huggingface.co/sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2/resolve/main/pytorch_model.bin" \
    -o pytorch_model.bin

Create seed script at artifacts/api-server/src/scripts/seedProducts.ts

Generate exactly 25,000 products across these 13 categories (~1,900 per category):
  Electronics, Mobile Phones, Laptops & Computers, Audio & Headphones,
  Cameras, Men's Clothing, Women's Clothing, Shoes & Footwear,
  Watches & Jewelry, Home & Kitchen, Furniture, Beauty & Skincare,
  Sports & Fitness

For each product:
  - name: realistic English name (e.g. "Samsung Galaxy S24 Ultra 256GB")
  - name_ar: realistic Arabic name (e.g. "سامسونج جالاكسي S24 الترا 256 جيجا")
  - description: 2-3 sentence Arabic description
  - price: realistic USD price per category
      phones: $200–$1200, laptops: $400–$2000, clothing: $15–$200,
      furniture: $50–$800, beauty: $10–$150, sports: $20–$500
  - stock: random 0–200
  - category: one of the 13 categories
  - seller_id: query real sellers: SELECT id FROM users WHERE role='seller' LIMIT 5
    Distribute randomly across available sellers
  - image_url: https://picsum.photos/seed//400/400

Insert in batches of 500 with try/catch per batch.
Log progress every 2,500: "Seeded 2500/25000 (10%)"

Add to package.json: "seed:products": "npx tsx src/scripts/seedProducts.ts"

Run it: cd artifacts/api-server && npm run seed:products
