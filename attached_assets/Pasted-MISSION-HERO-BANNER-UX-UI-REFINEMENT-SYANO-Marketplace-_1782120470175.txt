====================================================
OBJECTIVE
====================================================

Improve the homepage Hero Banner to behave like modern marketplace platforms (Amazon, Noon, Trendyol, Alibaba).

The current banner has several UX issues:

1. Arabic text is positioned incorrectly.
2. Navbar visually overlaps the banner from the top.
3. Banner height is too large and wastes vertical space.
4. Navigation arrows are outdated and visually noisy.
5. Hero content is not aligned properly for RTL.
6. Mobile experience is not optimal.
7. Banner should feel like a modern marketplace, not a landing page.

====================================================

Current banner is too tall.

Desktop:
