STEP 5 — Start Embedding Service
Start the embedding service:
cd artifacts/embedding-service && uvicorn main:app --host 0.0.0.0 --port 8001 &
Wait 45 seconds for model to load (transformer) or 5 seconds (TF-IDF).
Then verify it is running:
curl http://localhost:8001/health
Expected response:
json{ "status": "ok", "model": "multilingual-e5-small", "vector_dimensions": 384 }
Then test Arabic embedding:
curl -X POST http://localhost:8001/embed/query \
  -H "Content-Type: application/json" \
  -d '{"text":"هاتف ذكي سامسونج"}'
Expected: embedding array of exactly 384 numbers, processing_ms logged.
Then test English embedding:
curl -X POST http://localhost:8001/embed/query \
  -H "Content-Type: application/json" \
  -d '{"text":"samsung smartphone"}'
Expected: embedding array of 384 numbers — DIFFERENT from Arabic but semantically similar.
Then test batch:
curl -X POST http://localhost:8001/embed/batch \
  -H "Content-Type: application/json" \
  -d '{"texts":["هاتف","لابتوب","فستان"],"type":"passage"}'
Expected: 3 embeddings, each exactly 384 dimensions.
If any test fails: fix before continuing. Do not proceed with broken embedding service.
