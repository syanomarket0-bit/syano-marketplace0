Find every implementation.

Determine whether:

## Create One Shared Utility

Create a single utility:

calculateDiscountPercent()

Example:

originalPrice = 1131
salePrice = 1078.97

Discount:

((1131 - 1078.97) / 1131) * 100

Then use ONE standardized output.

Rules:

* round to nearest whole number
* never show decimals
* never show conflicting values

Result:

Everywhere.

No exceptions.
