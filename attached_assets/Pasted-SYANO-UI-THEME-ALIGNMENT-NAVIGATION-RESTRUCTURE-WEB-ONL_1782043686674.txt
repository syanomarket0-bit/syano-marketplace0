# PART 1 — GLOBAL THEME COLOR ENFORCEMENT (CRITICAL)

The entire web system must strictly follow SYANO design tokens.

## RULES:

1. Replace ALL hardcoded colors with theme tokens:

   * NO hardcoded green (#00xx or similar)
   * NO inline color styles
   * NO arbitrary Tailwind colors outside theme system

2. Use ONLY:

   * var(--primary)
   * var(--primary-foreground)
   * var(--background)
   * var(--foreground)
   * var(--muted)
   * var(--border)

3. Ensure consistency across:

   * Buttons
   * Sidebar active states
   * Cards
   * Badges
   * Inputs
   * Tables
   * Status indicators

## PRIMARY COLOR RULE

SYANO primary color must be the ONLY accent color used across UI.

All green accents must be unified into:

→ var(--primary)

## TEXT COLOR RULE

All text must follow theme rules:

* Primary text → var(--foreground)
* Secondary text → var(--muted-foreground)
* Buttons on primary backgrounds → var(--primary-foreground)

NEVER use:

* black/white hardcoded
* gray-900 / gray-100 outside theme mapping
