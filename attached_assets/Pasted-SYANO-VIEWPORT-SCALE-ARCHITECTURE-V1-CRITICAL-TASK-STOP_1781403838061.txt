REAL OBJECTIVE

The entire SYANO interface should behave like a professionally engineered scalable canvas.

When viewport size changes:

The page should remain visually identical.

The structure should remain identical.

The hierarchy should remain identical.

The alignment should remain identical.

The composition should remain identical.

Only the SCALE should change.

Think:

Zooming out on a Figma file.

Zooming out on a design system.

The user should feel they are looking at the same page through a different zoom level.
