====================================================
PHASE 3 — GUEST MODE SUPPORT
============================

If user is not authenticated:

Move to Cart must:

1. Add product to guest cart
2. Remove product from guest wishlist
3. Update both counters instantly
4. Persist after refresh

No login required.

====================================================
PHASE 4 — AUTHENTICATED MODE SUPPORT
====================================

If authenticated:

Move to Cart must:

1. Add item to server cart
2. Remove item from server wishlist
3. Refresh both contexts
4. Update counters instantly

Use optimistic updates where appropriate.
