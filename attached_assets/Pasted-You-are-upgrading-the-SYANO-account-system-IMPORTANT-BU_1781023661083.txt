You are upgrading the SYANO (سوق سوريا) account system.

## IMPORTANT BUSINESS RULE

SYANO does NOT support multi-role accounts.

A user can only have ONE active account type at a time.

Allowed account types:

* customer
* seller
* courier
* admin

Examples:

✅ customer → seller

✅ customer → courier

✅ seller → customer (admin action)

✅ courier → customer (admin action)

❌ seller + courier

❌ customer + seller

❌ customer + courier

❌ seller + courier + customer

Never allow multiple roles simultaneously.

---

# GOAL

Implement a promotion workflow similar to the current seller application flow.

Every new account starts as:

```ts

```

Users can then request promotion to:

* seller
* courier

After approval, the account type changes.

No new account is created.

No second login is created.

No duplicate user record is created.

The same user is upgraded.
